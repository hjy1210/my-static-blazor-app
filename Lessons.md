## Getting Started
1. 用 [mslearn-staticwebapp-dotnet](https://github.com/MicrosoftDocs/mslearn-staticwebapp-dotnet)當模板，複製一個儲存庫 my-static-blazor-app 到 github 中。
2. 下載 my-static-blazor-app 到在地端
3. 用 VS 2022 開啟 my-static-blazor-app 裡面的 SoppingList 方案，內含Api, Client, Data三專案。
4. 執行 Client 專案
5. 程式正確執行，可是Program.cs裡面的中斷點失效。
